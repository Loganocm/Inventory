Full-Stack Inventory Management System
Overview
This project is a full-stack application built with React for the frontend and Express.js for the backend, designed to manage product inventory and locations. It allows users to view, add, edit, and delete products, as well as manage locations. The application is designed to be deployed on Heroku.

Features
User Authentication: Secure login and registration system using JWT authentication.
Product Management: Users can create, read, update, and delete products.
Location Management: Manage multiple inventory locations for products.
Responsive Design: The frontend is fully responsive, making it accessible on both desktop and mobile devices.
Error Handling: Global error handling for server-side issues.
Tech Stack
Frontend:

React.js
React Router
Axios for HTTP requests
CSS for styling
Backend:

Node.js
Express.js
MongoDB (via Mongoose)
JWT for authentication
Helmet for security
Morgan for logging
Cors for handling cross-origin requests
